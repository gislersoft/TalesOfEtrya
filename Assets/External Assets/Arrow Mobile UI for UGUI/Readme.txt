This is a mobile-friendly UI for Unity UI. 

* Requires Unity 5.0!

The product features:

- PSD Files 
- Vector Design (Scalable) 
- Sliced images (PNG, High Resolution)
- 11 button types
- 41 button icons 
- Fonts included 
- Custom scripts 
- Demo scenes 
- Prefabs 

The demo scenes are setup to scale with the screen size relative to 1920x1080.
The images are sliced from PSD files scaled up to 200%.

Contact me if you need any assistance or have a question. 
With best regards, Evil.

PS: To use the joystick script you must import the unity cross-platform input package and then import the joystick script.
